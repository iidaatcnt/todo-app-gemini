# TODOアプリ (Gemini CLI開発)

これは、Gemini CLIとの対話を通じて開発されたシンプルなTODOアプリケーションです。

## 機能
- タスクの追加
- タスクの削除
- タスクの完了/未完了の切り替え

## 開発の経緯 (プロンプト履歴)

### 1. プログラムの解析
**User:**
> ここにあるプログラムを解析してください

**Gemini:**
> (解析結果の要約)
> このプログラムは、シンプルな「TODOアプリ」です。
> - **主な機能**: タスクの追加、削除、完了
> - **ファイルの役割**:
>   - `index.html`: アプリの骨格
>   - `script.js`: アプリの動作
>   - `style.css`: アプリの見た目

**User:**
